Classify Environment



Written in: Python



This is an implementation of convolutional neural network with LeNet architecture using Keras with tensorflow as a backend. This algorithm classifies an input picture into 4 environment categories: forest, ocean, mountain, city.

Schedule to Calendar



Written in: HTML/CSS/JS



This Chrome Extension takes the schedule from a student's UC Davis schedulebuilder website and converts it to an ical format to import for a calendar application such as Google Calendar. This uses the Chrome Extensions API.

Schelling Segregation Model



Written in: C++ with SFML



This is a model of segregation based on a preference for neighbors being the same "type" or "race" as you. This model goes to show how even a slight preference can cause larger macroscopic segregation in population distribution.

Double Pendulum



Written in: Python



This is a double pendulum physics simulation which was created using pygame and pygame utilities. Displays chaos theory, that is, how even slight differences in initial positions can cause very different ending results.
